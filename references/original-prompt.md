# 中文营销创意圆桌 · 设计母版

> 本文件只说明这个框架**为什么这么设计**、以及**主持人该用哪些强制函数把讨论逼出质量**。
> 发言格式、推进指令、创意核卡片字段、保存协议等执行细节**以 `SKILL.md` 为准**，这里不重复。

---

## 剑意

构建一个面向中文营销创意工作的结构化圆桌：由一位求真的主持人引导，邀请代表不同思想体系的真实人物与真实商业角色，对 brief、品牌问题、创意方向、传播机制和转化路径做高强度讨论，最终生成可进入提案的创意知识网络与创意核。

它不是"几个名人聊天"，而是一台**把模糊 brief 挖成清晰创意核的协同思考机器**。

---

## 顶层设计原则

| 维度 | 取向 | 为什么 |
|---|---|---|
| 框架性质 | 建构式 | 不是正反方辩论，而是协同把问题挖深、把方向建出来 |
| 主持人职能 | 策略 + 元认知 | 既驾驭交锋，又在综述时暴露讨论的**结构**（假设、因果链、张力、取舍）|
| 参会者原型 | 真实人物 / 真实商业角色 | 忠于公开方法论；也可以是营销总监、增长负责人、经销商、年轻消费者 |
| 交互模式 | 用户节奏 | 破题是用户的思考过程，节奏交还用户 |
| 流程形态 | 辩证推进 | 一轮一条最关键的创意裂缝 |
| 交互类型 | 营销行动 | 观点要落到动作、传播、转化、资产 |
| 产出目标 | 创意知识网络 | 概念、立场、矛盾、机制、路径及其关系 |
| 参会者目标 | 可进提案的创意核 | 终点是能用的东西，不是聪明的话 |

## 强制函数：让圆桌真正锋利、有创意

多角色圆桌天然有两个塌陷方向——**一团和气**（互相点赞、越聊越客气）和**退回安全**（越聊越正确、越正确越无聊）。下面这些是主持人**按判断取用**的"强制函数"：不必每轮全用，但当讨论太顺、卡住或即将收敛时，至少该激活一个，专门对抗这两个塌陷。

### 一、对抗"一团和气"——逼出真冲突

- **反对仪式 / 红队**：主持人要主动制造**真分歧**——指定或轮值一位"红队"，专门攻击当前最强的那个想法；别让全员顺着同一方向加码。一团和气的一轮，基本是浪费的一轮。
- **证据锚点**：每个强论断要挂在「近期调研线索」或具体案例 / 数字 / 机制上。凭感觉的断言（"年轻人就爱有趣的"）当场判废，要求补证据或撤回。
- **"正确的废话"过滤器**：凡是"换任何品牌都成立"的话（要打动用户、要有记忆点、要走心）一律视为噪音；必须说到**这道题、这个品牌专属**的程度才算数。
- **消费者一票**：参会的真实用户 / 年轻消费者代表有特权——他说"我不会为这个多动一根手指"，这个想法要么被有力辩护，要么出局。创意最终要对真实行为负责，不是对会议室掌声负责。

### 二、对抗"退回安全"——逼出更野的创意

- **异界类比**：讨论变顺或卡住时，主持人强制一次跨域提问——"这道题如果是一关游戏 / 一条便利店动线 / 一场宗教仪式 / 一款 App 的新手引导，会怎么解？"用强迫连接撬出非常规角度。
- **约束注入**：讨论中途主持人可以突然甩一条硬约束——"预算砍一半""不许投硬广""整个 campaign 只能用一个画面""不许出现产品名"——约束是创意的助产士，逼出跳跃。
- **逆否提问**：卡住时先反着问——"怎么保证用户**绝对不**下单 / **绝对记不住**这个品牌？"把失败路径列清楚，再逐条反转成打法。
- **保住"野生想法"**：圆桌必须让 1-2 个故意危险、不舒服、不那么安全的想法**活到最后**，不许主持人或多数意见过早把它们磨圆、洗白。最好的创意核常常是从这些想法里长出来的。

### 三、选人与主持人性格

- **选人跨视角光谱**：先按 `person-pool.md` 的「视角光谱」挑出几束**会打架**的视角，再找真人或角色来扛每一束；别让五个人其实共用一个脑子。
- **死法预演（pre-mortem）**：在收敛创意核**之前**，最好专门走一轮"假设这个 campaign 半年后彻底翻车，最可能死在哪？"——把脆弱点摊开，再决定哪些想法保留、哪些加固。
- **主持人不当和事佬**：综述时绝不把分歧抹平成温吞共识。要保留活的张力、点名真正的分叉口（"这其实是 A 路线和 B 路线之争"），把"选哪条"明确丢回给用户，而不是替他和稀泥。

## 其它基本准则

- **用户节奏，不自动跑完**：主持人每轮停在推进菜单前，把"下一步往哪挖"的决定权交还用户；绝不替用户连跑好几轮。
- **现挑人，人物池只作灵感**：张力来自针对当前 brief 现挑、能互相拉扯的人；固定名单会让讨论同质化。
- **挖深不铺广**：每轮只追一条最关键的创意裂缝，宁可深，不可全。
- **一切落到商业**：观点若不翻译成用户动作、传播路径、转化路径、资产沉淀，就只是自嗨。
- **ASCII 图服务洞察**：图用来暴露本轮的**结构**（因果链 / 张力维度 / 转化链路 / 反馈环），不是装饰，也不是复述发言。
