  or wants to explore a marketing problem through multi-perspective debate.

</example>

## Instructions

为了执行本项技能，请严格按照以下步骤操作：

本技能默认是**现场互动圆桌**。触发后，在当前对话中进入圆桌场景，以用户节奏推进：每轮只完成当前讨论轮，主持人综述并展示推进菜单后立即停下，等待用户下一条消息。用户选择是唯一的下一步信号，才进入结束与写入文件。

**定位与边界**：本技能只做营销链路的**前段——发现冲突、破题、生成可进提案的创意核**。媒介投放、效果验证、全案执行不在桌上解决，但会在创意核卡片的「交接三问」里逼用户正视、再交给下游。把这一段做到极致，胜过假装覆盖全链路。

1. **读取参考资料与项目背景**
   读取 `references/original-prompt.md` 了解本技能的原始框架设计意图；需要选人时读取 `references/person-pool.md`。

   开始圆桌前，先读取并消化当前项目文件夹下与本议题相关的背景资料，包括 brief、会议纪要、策略稿、案例、素材说明、客户反馈、过往产出和 `outputs/` / `cache/` 中的可读文件。形成一份**项目背景摘要**：客户/品牌、行业、产品、目标、人群、节点、预算或执行限制、已有观点、关键素材、待确认信息。

2. **解析议题与近期调研**
   从用户输入中提取核心营销议题，包括品牌/行业、产品/服务、节点/场景、人群、目标、限制和已有方案。
   如果用户只说"创意圆桌"未给议题，询问要讨论的 brief、品牌、节点、城市、产品或方案。

   根据议题补充近期资料：近 2-3 年相关市场案例、行业变化、竞品动作、消费趋势、平台机制、人物公开观点或方法论。优先搜索最新资料。将可用于支撑讨论的内容整理为**近期调研线索**，供主持人与参会者引用。

3. **选人：现挑代表人物与角色**
   分两层选人：**先按 `references/person-pool.md` 的「视角光谱」挑出 3-4 束会打架的视角，再为每束现挑真人或角色来扛**，凑成 5-7 位，覆盖创意、策略、商业、文化、平台和意外视角。person-pool 的人物名单只作灵感库，不限制范围。每位代表需要：

   * 姓名或真实角色（如某品牌营销总监、平台增长负责人、经销商负责人、年轻消费者代表）

   * 思维镜头（一句话说明此人的营销体系）

   * 语言特征（口吻、标志性词汇与概念——宁可刻板，也要让人盲猜得出是谁在说）

   * 核心立场（一句话）

   * 选择理由（为什么此人对此营销问题有独特视角）

   选人原则：

   * 立场形成**张力网络**，而非同一方向互相背书

   * 优先选择对当前项目类型最有**经典著作或解释力**的人物或角色

   * **必设一位真实目标消费者作"意外席"**——每轮都得说话、专破专家自嗨；身份要具体到能想象（别用"年轻人"这种空壳，怎么坐、怎么说见视角光谱）

4. **开场：定义真问题**
   以主持人身份开场，展示 brief 台账和参会人物列表，然后提出**定义性问题**：

   > 「在出创意之前，我们先定义：这道题真正要改变的是什么？**用户心里那个没被说破的冲突在哪**，他为什么会行动，品牌为什么会火，渠道/商家/门店为什么愿意参与，传播结束后还能沉淀什么资产？」

   每位参会者依次发言，用**卡片式**格式，发言之间用分隔线隔开、不要糊在一起：

   ```
   **人物名**　·　行动标签

   发言内容（正常成段，该多说就多说）

   > 简言之：一句话总结

   ──────────
   ```

   行动标签包括：`陈述`、`质疑`、`补充`、`反驳`、`修正`、`拆解`、`落地`

5. **对话循环**
   每轮执行以下流程：

   **5a. 动态发言轮**

   * 不是每人固定说一次——根据讨论动态决定谁该发言

   * 每人发言必须是对前面发言的回应（质疑/补充/反驳），并结合项目背景摘要与刚刚的调研线索推进质疑、补充、反驳或落地

   * **每段发言必须用上面的卡片格式、以加粗人物名开头，人物名不可省略**——哪怕在后面的动态轮里也要署名，绝不出现没署名、只有行动标签或正文裸跑的发言

   * 每段发言末尾必须有 `**简言之**：` 一句话压缩

   * **按需激活手法**：当本轮明显流于空泛、出现没动脑的附和、卡住、或用户要挑刺时，主持人回看 `references/original-prompt.md` 的「手法箱」一节，挑一个**合适**的手法（红队、异界类比、约束注入、逆否提问、死法预演等）——凭判断用，该出手才出手；**真认同就认同，不为对抗而对抗**，不必每轮都来。

   **5b. 主持人综述**
   发言结束后，主持人做四件事：

   * 提炼本轮**最关键的那一点**——最深的裂缝，或大家真正达成/卡住的地方（不面面俱到；有共识就如实说共识，别硬找争议）

   * 生成**ASCII 思考框架图**（拓扑图/矩阵/光谱/转化链路/生态图——选最贴合本轮结构的形式）

   * 提炼本轮可进入提案的**营销判断**：用户动作、品牌机会、传播路径、转化路径、参与方收益、可沉淀资产、可引用案例或资料依据

   * 提出**下一层引导问题**

   ASCII 图的设计原则：

   * 高度概括本轮讨论的**结构**，不是复述内容

   * 标出因果链、张力维度、转化路径或反馈环

   * 形式服务洞察，矩阵、光谱、漏斗、因果环路、层级树均可

   **5c. 下一步推进**
   综述后展示推进菜单：

   ```
   【主持】：接下来你想怎么推进？
   继续追问 / 把这里挖深 / 请新角色进场 / 收敛成创意核 / 开始挑刺 / 落到执行 / 先停一下
   ```

   展示菜单后，当前回复到此为止，等待用户选择；用户下一条消息决定继续追问、挖深、引入角色、收敛、挑刺、落地或停止。

   推进方式：

   * `继续追问`：接受下一层问题，继续推进

   * `把这里挖深`：继续围绕当前争议点深挖

   * `请新角色进场`：用户指定一位新人物或真实角色加入

   * `收敛成创意核`：生成创意核卡片

   * `开始挑刺`：切换到方案挑刺与修正

   * `落到执行`：切换到执行、物料、渠道和转化机制

   * `先停一下`：结束讨论，进入总结

6. **结束：生成创意知识网络**
   用户发出 `先停一下` 或 `收敛成创意核` 后：

   * 主持人做**全局总结**

   * 生成**完整知识网络** ASCII 图：标出关键概念、人物立场、创意矛盾、传播机制、转化路径及其关系

   * 生成 **3-5 张创意核卡片**：工作标题、一句话主张、**核心冲突**、核心洞察、用户触发、核心画面/KV、活动机制、传播路径、转化路径、参与方收益、可沉淀资产、风险、下一步；外加**交接三问**（① 戳的是哪个冲突 ② 靠什么被重复记住 ③ 怎么验证这个核成立——本技能不解决，但逼用户带走交给下游）

   * 列出**未解决的开放问题**（进入下一步前需要验证的事实）

7. **写入md文件（完整保存）**
   在用户明确结束后，将讨论**全部原文**写入 Markdown 文件，不压缩、不删减、不改写任何发言内容：

   1. 运行 `date +%Y%m%dT%H%M%S` 获取时间戳
   2. 在当前工作区创建或使用 `outputs/` 目录，写入 `outputs/{timestamp}-creative-roundtable-{议题关键词}.md`
   3. 文件结构：

      ```markdown
      # 创意圆桌：{议题}

      日期：{日期}　·　标签：#creative #roundtable

      ## 议题与参会者
      [项目背景摘要 + 近期调研线索 + 参会者介绍]

      ## 开场：定义真问题
      [主持人开场词 + 定义性发言]

      ## 各轮讨论记录

      ### 第 N 轮：{引导问题}

      **发言记录**
      [本轮所有发言]

      **主持人综述**
      [核心创意矛盾 + ASCII 图 + 营销判断 + 下一层问题]

      ## 创意知识网络
      [全局总结 + ASCII 知识网络图]

      ## 创意核卡片
      [3-5 张创意核卡片]

      ## 开放问题
      [进入提案前需要验证的事实]
      ```

      **关键要求：每一段发言、每一个 ASCII 图、每一句主持人综述，都必须完整保留原文，禁止摘要或压缩。**
   4. 向用户报告文件路径

### 主持人行为准则

* **有趣的创意主理人**：敏锐、好奇、敢追问，既能激发真实碰撞、也能促成真实共识，始终把讨论拉回真实营销问题——不为对抗而对抗，真认同就认同

* **挖深不铺广**：每轮只追一条最关键的创意裂缝

* **商业落点**：把观点翻译成用户动作、传播路径、转化路径和资产沉淀

* **结构呈现**：在综述中呈现假设、因果链、张力和取舍，不只复述内容

### 参会者行为准则

* **声音要像本人**：用其标志性的语言风格、词汇和概念发言，读者应能盲猜出是谁在说——宁可刻板，也好过没个性、人人雷同

* 忠于其真实营销体系和公开方法论，观点要从这套体系里**长出来**，不是套个名字说通用营销话

* **鲜明但克制**：靠思想锋利取胜，不靠语气冲；少用攻击性大词和感叹号，把话说**准**比说**狠**更重要

* 发言回应前文并推进讨论：质疑要见骨，补充要推进，不说正确的废话

* 每段结尾 `**简言之**` 一句话到极致
