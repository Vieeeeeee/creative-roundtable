# creative-roundtable Skill Creator 测试报告

日期：2026-06-07

## 测试目标

验证 `creative-roundtable` 是否成为“中文营销创意通用圆桌”，并且拥有自己的独立运行母版。

## 成功标准

- 开始圆桌前先读取并消化项目文件夹下的 brief、会议纪要、案例、素材说明、客户反馈和已有产出。
- 能补充近 2-3 年相关市场案例、行业变化、竞品动作、消费趋势、平台机制和人物公开观点。
- 每轮人物发言都要针对项目背景和近期调研线索，不做脱离项目的泛聊。
- 触发后进入当前对话中的现场互动圆桌；每轮展示推进菜单后停下等待用户选择，不自动跑完整场，也不提前写文件。
- 选人从全世界真实人物与真实商业角色中现挑，人物池只做灵感库；可加入品牌营销总监、平台增长负责人、门店/渠道角色和年轻消费者代表。
- 能被 `$creative-roundtable`、`创意圆桌`、`大咖圆桌`、`让大咖来辩`、`营销创意`、`slogan`、`KV`、`审稿`、`打磨方案` 等触发。
- 默认输出中文，适合中文营销提案、洞察、文案和调研表达。
- 国外大咖不被“中文营销黑话化”，而是输出其公开思想体系、价值观和判断逻辑。
- 适用于国内互联网公司、大型实业公司、消费品牌、平台业务、城市文旅、本地生活、汽车家电、快消和 B2B/产业项目。
- 不绑定淘宝闪购；淘宝闪购只作为当前项目上下文被读取。
- 最终能收敛到创意核、转化路径、渠道/商家/门店参与、传播路径、资产沉淀、风险和 48 小时下一步。

## 测试用例结论

### 1. 本地生活平台夜宵季

Prompt：`创意圆桌：一个本地生活平台要做暑期夜宵季 campaign，希望吸引年轻人下单，也让商家愿意参与，帮我破题。`

结论：通过。metadata 和正文均会触发；会进入破题圆桌。质量过滤器覆盖用户动作、商家收益、平台资产、转化路径，适合该场景。

### 2. 国产新能源车家庭试驾

Prompt：`让大咖来辩：一个国产新能源车品牌想做一次家庭用户试驾传播，不想只讲参数，怎么做得有信任感？`

结论：通过。description 覆盖大型实业、汽车家电；会按行业现挑真实人物或真实商业角色；不会默认套淘宝闪购逻辑。

### 3. Slogan/KV 审稿

Prompt：`帮我审稿这个 slogan 和 KV 方向：'城市醒来，早餐准时到'，客户是一个即时零售平台。`

结论：通过。description 覆盖 slogan、KV、审稿；工作模式包含审稿圆桌和命名圆桌；会对画面感、转化路径、套话风险做检查。

### 4. 淘宝闪购区域比稿

Prompt：`淘宝闪购区域比稿，想做一个城市专属 campaign，创意圆桌先帮我找几个创意核。`

结论：通过。当前项目可读取淘宝闪购约束；正文明确“这些限制只服务当前项目”。

## 静态断言结果

- 中文优先：通过。
- 通用营销范围：通过。
- 不绑定淘宝闪购：通过。
- 独立运行母版：通过。`references/original-prompt.org` 已定义本 skill 自己的系统定位、顶层原则、角色系统、选人协议、对话循环、质量过滤器、收敛产物和保存协议。
- 提案可用收敛：通过。
- 现场互动节奏：通过。

## 仍可继续增强

- 可以再补一个 `creative-research` skill，和本圆桌形成“先调研提示词，后圆桌破题”的组合。
- 可以未来增加一份真实输出样例库，例如汽车、白酒、本地生活、城市文旅各一条，帮助模型更稳定地拿捏语气和深度。
